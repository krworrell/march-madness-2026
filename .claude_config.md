## Architecture Notes
- **tournament_tracker.html** is a single self-contained HTML file (no dependencies except Chart.js CDN). All data is embedded in JS objects at the top of the `<script>` block.
- **Bracket state** uses a flat picks model: `{region}-{round}-{slot}` keys mapping to team names. Stored in localStorage key `calcutta2026_picks`.
- **To update actual results:** Modify the `BRACKET` data or add a `CONFIRMED` results object in the JS. The "Won to Date" column in the portfolio table is designed to be populated with actual payouts as rounds complete.
- **Team names with apostrophes** (Saint Mary's, St. John's) are escaped via `esc()` function in onclick handlers.
- The pre-auction dashboard uses a separate localStorage key `calcutta2026_state`.

## Auction Context
- Hosted on Victory's Auction Pro (auctionpro.co)
- Teams auctioned in random order, open ascending bid
- 10-11 buying groups historically, ~9 this year
- Pot = sum of all winning bids (no house take)
- Kenny's valuation methodology: 4-source composite (KenPom, Massey, BPI, Silver Bulletin/538) averaged for round-by-round win probabilities, then multiplied by payout structure
